# UniKey — eCommerce Keychain Website

## Project Overview

UniKey is a mini e-commerce website built as part of a college curriculum project. The site showcases keychain categories and product pages, and it includes a simple order workflow using HTML, CSS, Bootstrap, JavaScript, and PHP.

## What this project includes

- Responsive landing/home page with product category cards
- Product collection pages for Fabric, Leather, Wood, Silicon, Glass, and Cotton keychains
- Working Bootstrap carousel on the home page
- Simple order page with form inputs and PHP-based order submission capability
- Contact page with PHP server-side form handling

## Tools and technologies

- HTML for page structure
- CSS for custom styling
- Bootstrap for responsive layout and components
- JavaScript for UI behavior and form handling
- PHP for backend form submission and processing

## Notes

This project is intended as a mini college project, demonstrating a basic e-commerce front-end with PHP integration. The code is organized to keep styling and content separate, using Bootstrap for layout and visual consistency.

## How to run

1. Place the project in a PHP-enabled web server environment (XAMPP, WAMP, or similar).
2. Open `home.html` in the browser or load the project root in the server.
3. Use the navigation links to explore categories, product pages, and contact/order flows.
